# Urge Surfer

> Cravings are waves. They rise, they peak, they pass. This app teaches you to ride them out instead of fighting them.

## Why I built this

Most "addiction" apps shame you for feeling the urge. Streak-broken alarms, red banners, abstinence calendars. The message under the surface is _don't feel it_.

But you can't will an urge away. What you _can_ do is wait it out — every craving has a half-life, usually under five minutes if you don't act on it. Mindfulness research calls this **urge surfing**: noticing the wave, breathing through it, watching it crest, watching it pass.

I wanted a tool that did one thing well: when you feel the pull — cigarette, coffee, alcohol, doomscroll, whatever — you open the app, name the urge, set a timer, and ride it out. If you make it through, the app celebrates with a victory selfie. If you cave, it asks what triggered it, no shame, and feeds the data into stats so you start to see your own patterns.

It's the app I wished existed.

## What it does

- **Pick what you're surfing** — 12 presets (cigarette, vape, alcohol, coffee, sugar, weed, social media, shopping, porn, gambling, snacking, nail biting), plus add your own.
- **Set a timer** — 1, 3, 5, 10, or 15 minutes. Most waves pass before the bell.
- **Ride it out** — animated breathing circle, 4-7-8 cycle prompts, intensity slider you can adjust mid-surf, timer that survives backgrounding _and_ a force-quit.
- **Or cave** — one tap. The app asks _what pulled you under_ (Stress, Boredom, Social, After meal, Habit, Alcohol, Anger, Reward), logs it, and offers a retry.
- **Victory selfie** — capture the moment, pick from five overlay styles (Minimal, Strava, Ticker, Polaroid, Receipt), save to a private album, share if you want.
- **See your patterns** — heatmap by day-of-week × hour, top triggers ranked by frequency, success rate per urge type.
- **Honest streak** — days since your last cave for that urge, not "consecutive successful logs" you have to remember to file every day. Silence doesn't break it.

## Tech

- **React Native 0.76** + **Expo SDK 52**, plain JavaScript
- **AsyncStorage** for everything — no backend, no account, no analytics, no crash reporting
- Wall-clock timer using an absolute `endTime`, recomputed on every tick + on `AppState` foreground, so backgrounding or killing the app never breaks the surf
- Light/dark theming via React context + a CSS-var-style palette, system color scheme as default
- `expo-image-picker` for capture, `expo-sharing` for the share sheet, `react-native-svg` for the breathing circle

The app does not call home. Name, tracked urges, log, and photos all live in your device's sandbox only. See [SECURITY.md](SECURITY.md) for the threat model.

## Run it locally

You'll need Xcode (for the iOS simulator) and Node 20+.

```sh
git clone https://github.com/abhaykatheria/urgesurfer.git
cd urgesurfer
npm install
npx expo start --ios
```

Press `i` to open the iOS simulator, or scan the QR with Expo Go on your phone.

For a native build (e.g. to install on a real device without Expo Go, or to test camera permissions on hardware):

```sh
npx expo prebuild --clean
npx expo run:ios
```

The `ios/` and `android/` folders are generated by `expo prebuild` and intentionally git-ignored — they're regenerated from `app.json` whenever you build.
